# Project Genesis

Project Genesis is Raiatea's Inception phase under a lean, risk-driven Unified Process.

## Intended documents

- `00-why-raiatea.md` — the problem and the reason for beginning.
- `01-manifesto.md` — the human and philosophical commitments.
- `02-vision.md` — the long-term direction and measurable value.
- `03-system-context.md` — actors, environment, boundaries, and external systems.
- `04-product-map.md` — shared core and possible products.
- `05-use-case-model.md` — primary user goals and architecturally significant scenarios.
- `06-risk-list.md` — existential, technical, legal, scientific, and product risks.
- `07-glossary.md` — an initial ubiquitous language.
- `08-inception-review.md` — evidence required before entering Elaboration.

These files will be created only as their questions are genuinely addressed. Empty documentation is not progress.
