# Project Genesis

Project Genesis is Raiatea's Inception phase under a lean, risk-driven Unified Process.

Its documentation has two complementary layers:

- the numbered files in this directory are **exploratory notes** that preserve
  the reasoning, alternatives and long-term research behind Raiatea;
- [`inception/`](inception/) contains the **canonical Inception artifacts**
  used to orient current decisions and development.

Exploratory notes are sources, not automatically accepted decisions. Canonical
artifacts must cite the notes from which they derive and must distinguish
implemented behavior, accepted decisions, hypotheses, plans and deferred
research.

## Canonical Inception artifacts

The canonical index, document lifecycle and assertion-status vocabulary are
maintained in [`genesis/inception/README.md`](inception/README.md).

The intended artifacts are:

- `00-why-raiatea.md` — the problem and the reason for beginning.
- `01-manifesto.md` — the human and philosophical commitments.
- `02-vision.md` — the long-term direction and measurable value.
- `03-system-context.md` — actors, environment, boundaries, and external systems.
- `04-product-map.md` — shared core and possible products.
- `05-use-case-model.md` — primary user goals and architecturally significant scenarios.
- `06-risk-list.md` — existential, technical, legal, scientific, and product risks.
- `07-glossary.md` — an initial ubiquitous language.
- `08-inception-review.md` — evidence required before entering Elaboration.

The canonical directory also contains an audit created before these artifacts:

- [`00-genesis-audit-and-traceability.md`](inception/00-genesis-audit-and-traceability.md)
  — mapping of the founding conversation, exploratory notes, documentation gaps
  and implementation state observed on 21 July 2026.

The audit does not shift or replace the intended artifact names. The canonical
index assigns their final paths as they are created, avoiding placeholder files
that suggest questions have already been answered.

## Exploratory notes

The notes from `00-understanding.md` through
`20-pedagogical-visual-standard.md` remain at their current paths so historical
links and provenance are preserved. Their complete linked legend is available
in the [Genesis audit](inception/00-genesis-audit-and-traceability.md#21-legenda-delle-fonti-genesis).

These files are created or promoted only when their questions are genuinely
addressed. Empty documentation is not progress.
