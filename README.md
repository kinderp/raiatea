# Raiatea

*A private research project for navigating evolving knowledge with provenance, context, and human judgment.*

Raiatea is currently in **Project Genesis**. The repository is intentionally private while its vision, language, ontology, and first verifiable vertical slice are developed.

## Prototype decisions

The pedagogical-module vertical slice records implementation-specific decisions under [`prototype/pedagogical-module/docs/`](prototype/pedagogical-module/docs/). The stable identity and revision contract for future module evolution is documented in [`stable-step-identity.md`](prototype/pedagogical-module/docs/stable-step-identity.md). It keeps learner-evidence v1 frozen while defining durable module IDs, explicit revisions, stable conceptual step IDs, compatibility classes, and reviewed migration responsibilities.

## Current phase

**Leaving the Shore** — establishing the project's purpose, principles, vocabulary, boundaries, and research method before implementation.

No public claims are made yet about what Raiatea will become. Results must precede declarations.
